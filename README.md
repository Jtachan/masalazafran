# Masala & Azafran

International recipes cooked from an indian-spanish kitchen.

## Categories

All recipes are stored at the `recipes/` folder.
Within it, there are subfolders defining each one a sub-category.

Each sub-category defines the origin of each recipe.
E.g.: spanish, indian, mexican, french, etc. 

## Recipe format

The recipes are written all in pure markdown, defined with a specific structure:

```markdown
---
tags:
  - TagA
  - TagB
---

# Recipe name

A description and/or introduction of the recipe.

![Image of the recipe]

---

**Ingredients**

- _Ingredient A_ (200g)
- _Ingredient B_ - with some comment

**Veg. Alternative**

- _Vegan ingredient C_ substituting ingredient B

---

**Steps**

1. Reading the recipe
2. Preparing the recipe
```
